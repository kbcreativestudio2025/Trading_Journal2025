How Data is Stored and Managed

Local Storage

All trade data is stored in your browser's localStorage. This means:

  • Data persists even after closing the browser
  • Accessible only on this device/browser
  • Survives page refreshes

Backup Function

The Backup button:

  • Creates a JSON file with all your trade data
  • Allows you to save this file externally
  • Protects against accidental data loss
  • Useful when changing devices

Restore Function

The Restore button:

  • Uploads a previously saved JSON file
  • Replaces current data with backup data
  • Useful when setting up on a new device
  • Recovers data after browser reset

Data Safety Tips

1. Regularly backup your data using the Backup button

2. Store backup files in multiple locations (cloud, USB, etc.)

3. Before clearing browser data, ensure you have a recent backup

4. When switching devices, restore from your latest backup file
